**Time:** 9 hours ago

**Issue:** After enabling Gemini API, previously public Google Maps API keys can access sensitive endpoints.

**Impact:**
- 2,863 API keys found that can access Gemini
- Includes Google's own keys (earliest deployed in February 2023)
- May lead to unauthorized access and billing charges

**Recommendation:** Immediately check and rotate API keys, separate keys for different services.
