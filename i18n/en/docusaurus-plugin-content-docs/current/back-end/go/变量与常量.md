# Variables and Constants - Go's Declaration Style

## 📦 Variable Declaration

### 1. Standard Declaration

```go
var age int
age = 25

var name string
name = "John"
```

**Syntax:** `var variableName type`

**Zero Values:**
| Type | Zero Value |
|------|-----------|
| Numeric | `0` |
| Boolean | `false` |
| String | `""` |
| Pointers/Interfaces | `nil` |

### 2. Short Declaration (Recommended)

```go
age := 25        // int
name := "John"   // string
salary := 9999.99 // float64
```

**Most commonly used in Go!**

### 3. Multiple Variables

```go
var (
    name   = "John"
    age    = 25
    salary = 9999.99
)
```

## 🔄 Variable Swap

```go
a, b := 10, 20
a, b = b, a  // No temporary variable needed!
```

## 🔒 Constants

```go
const pi = 3.14159
const maxUsers = 1000
```

### iota Enumerator

```go
const (
    Sunday = iota    // 0
    Monday           // 1
    Tuesday          // 2
    // ...
)
```

## 📊 Comparison

| Feature | Variables | Constants |
|---------|-----------|-----------|
| Modifiable | ✅ Yes | ❌ No |
| Keyword | `var` / `:=` | `const` |
| Type Inference | ✅ Yes | ✅ Yes |
