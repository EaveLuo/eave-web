# Testing - 单元Testing、基准Testing与模糊Testing

Go 内置强大的Testing工具，无需第三方库。

## 🧪 Testing基础

### Testing文件命名

```
calculator.go        # 源代码
calculator_test.go   # Testing文件
```

### 基本Testing结构

```go
package calculator

import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    expected := 5
    
    if result != expected {
        t.Errorf("Add(2, 3) = %d; expected %d", result, expected)
    }
}
```

### 运行Testing

```bash
go test              # 运行所有Testing
go test -v           # 详细输出
go test -cover       # 覆盖率
go test -bench=.     # 基准Testing
```

## 📋 表格驱动Testing

```go
func TestAdd(t *testing.T) {
    tests := []struct {
        name     string
        a, b     int
        expected int
    }{
        {"正数相加", 2, 3, 5},
        {"负数相加", -2, -3, -5},
        {"零相加", 0, 0, 0},
    }
    
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            result := Add(tt.a, tt.b)
            if result != tt.expected {
                t.Errorf("Add(%d, %d) = %d", tt.a, tt.b, result)
            }
        })
    }
}
```

## ⏱️ 基准Testing

```go
func BenchmarkAdd(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Add(2, 3)
    }
}
```
