# Standard Library - Essential Packages

Go's standard library is called "Batteries Included". Here are the most commonly used packages.

## 📦 Core Packages

### I/O and Formatting

| Package | Purpose | Example |
|---------|---------|---------|
| `fmt` | Formatting I/O | `fmt.Printf`, `fmt.Sprintf` |
| `io` | I/O primitives | `io.Reader`, `io.Writer` |
| `bufio` | Buffered I/O | `bufio.Scanner` |
| `strings` | String operations | `strings.Join`, `strings.Split` |

### Data Encoding

| Package | Purpose | Example |
|---------|---------|---------|
| `encoding/json` | JSON processing | `json.Marshal`, `json.Unmarshal` |
| `encoding/xml` | XML processing | `xml.Marshal` |
| `encoding/base64` | Base64 encoding | `base64.StdEncoding` |

### Network and HTTP

| Package | Purpose | Example |
|---------|---------|---------|
| `net/http` | HTTP client/server | `http.Get`, `http.HandleFunc` |
| `net/url` | URL parsing | `url.Parse` |

### Time

```go
import "time"

now := time.Now()
formatted := now.Format("2006-01-02 15:04:05")
```

## 🔤 fmt Formatting

```go
fmt.Printf("%v\n", 42)        // 42 (default)
fmt.Printf("%T\n", 42)        // int (type)
fmt.Printf("%d\n", 42)        // 42 (decimal)
fmt.Printf("%s\n", "hello")   // hello (string)
```

## 🌐 net/http Example

```go
// Server
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello!")
})
http.ListenAndServe(":8080", nil)

// Client
resp, _ := http.Get("https://api.example.com")
defer resp.Body.Close()
```
