# Basic Data Types - Numbers, Strings, Booleans

Go's type system is concise yet powerful. Let's dive into basic data types.

## 🔢 Numeric Types

### 1. Integer Types

Go provides rich integer types:

```go
// Signed integers
var age int8    // -128 to 127
var level int16 // -32768 to 32767
var score int32 // -2 billion to 2 billion
var money int64 // Very large integers

// Unsigned integers
var count uint8  // 0 to 255
var port uint16  // 0 to 65535
var size uint32  // 0 to 4 billion
var big uint64   // Very large positive numbers

// Platform-dependent
var num int   // 32-bit on 32-bit systems, 64-bit on 64-bit systems
var size uint // Usually used for indices and counts
```

**Practical advice:**
- Default to `int` and `uint`
- Use `int32`, `int64` only when specific size needed
- Use explicit sizes for bitwise operations

### 2. Floating-Point Types

```go
var price float32  // Single precision, 6-7 significant digits
var pi float64     // Double precision, 15-16 significant digits

// Recommendation: Default to float64
price := 99.99
pi := 3.141592653589793
```

**Note:**
- `float32` has lower precision, use with caution
- For financial calculations, use `decimal` library (avoid precision issues)

### 3. Complex Number Types

```go
var c1 complex64  // float32 real + float32 imaginary
var c2 complex128 // float64 real + float64 imaginary

// Create complex number
c := complex(3, 4)  // 3 + 4i

// Built-in functions
real := real(c)  // 3.0 - get real part
imag := imag(c)  // 4.0 - get imaginary part
```

## 📊 Numeric Operations

### Basic Operations

```go
a, b := 10, 3

// Arithmetic operations
sum := a + b        // 13
diff := a - b       // 7
prod := a * b       // 30
quot := a / b       // 3 (integer division!)
rem := a % b        // 1 (modulo)

// Note: Integer division truncates
fmt.Println(10 / 3)   // 3
fmt.Println(10.0 / 3) // 3.333333
```

### Increment and Decrement

```go
count := 0
count++  // 1
count++  // 2
count--  // 1

// ❌ Error: Go doesn't have ++count or count--
// ++count  
// count--
```

**Important:** Go only has `count++` and `count--`, and they can only be standalone statements!

## 🔤 String Type

Strings are immutable sequences of bytes in UTF-8 encoding.

### Creating Strings

```go
// Double quotes - interpreted string
s1 := "Hello, World!"
s2 := "Line 1\nLine 2"  // Escape sequences work

// Backticks - raw string literal
s3 := `Raw string with \n no escapes`
s4 := `
    Multi-line
    string
`
```

### String Operations

```go
s := "Hello, Go!"

// Length (bytes, not characters)
len := len(s)  // 11

// Concatenation
s2 := s + " Language"  // "Hello, Go! Language"

// Indexing (returns byte)
b := s[0]  // 72 ('H')

// Slicing
sub := s[0:5]  // "Hello"

// Immutability
// s[0] = 'h'  // ❌ Error: strings are immutable
```

### Common String Functions

```go
import "strings"

s := "Hello, Go!"

// Contains
strings.Contains(s, "Go")     // true
strings.Contains(s, "Python") // false

// Prefix/Suffix
strings.HasPrefix(s, "Hello") // true
strings.HasSuffix(s, "!")     // true

// Index
strings.Index(s, "Go")     // 7
strings.Index(s, "Python") // -1

// Replace
strings.Replace(s, "Go", "Golang", 1)  // "Hello, Golang!"
strings.ReplaceAll(s, "l", "L")        // "HeLLo, Go!"

// Split/Join
parts := strings.Split("a,b,c", ",")  // ["a", "b", "c"]
joined := strings.Join(parts, "-")    // "a-b-c"

// Trim
strings.TrimSpace("  hello  ")        // "hello"
strings.Trim("...hello...", ".")      // "hello"
```

## ✅ Boolean Type

Booleans represent true or false values.

```go
var flag bool  // default: false

// Assignment
isValid := true
hasError := false

// Logical operations
a, b := true, false

and := a && b  // false (AND)
or := a || b   // true  (OR)
not := !a      // false (NOT)
```

### Comparison Operators

```go
a, b := 10, 20

eq := a == b   // false (equal)
ne := a != b   // true  (not equal)
lt := a < b    // true  (less than)
gt := a > b    // false (greater than)
le := a <= b   // true  (less or equal)
ge := a >= b   // false (greater or equal)
```

## 🔄 Type Conversion

Go requires explicit type conversion:

```go
var i int = 42
var f float64 = float64(i)    // Explicit conversion
var u uint = uint(i)
var s string = string(i)      // Converts to Unicode character '*'

// String to number
import "strconv"
n, _ := strconv.Atoi("42")    // string to int
f, _ := strconv.ParseFloat("3.14", 64)  // string to float64

// Number to string
s := strconv.Itoa(42)         // int to string
s := strconv.FormatFloat(3.14, 'f', 2, 64)  // float to string
```

**Important:** Go does NOT allow implicit type conversion!

```go
var i int = 42
var f float64 = i   // ❌ Error: cannot use i (int) as float64
var f float64 = float64(i)  // ✅ Correct
```

## 💡 Best Practices

### 1. Choose Appropriate Types

```go
// ✅ Good: Use int for general counting
for i := 0; i < 100; i++

// ✅ Good: Use specific sizes when needed
var port uint16 = 8080
var id int64 = 9999999999

// ✅ Good: Use float64 by default
var price float64 = 99.99
```

### 2. Avoid Magic Numbers

```go
// ❌ Bad: What does 86400 mean?
time.Sleep(86400 * time.Second)

// ✅ Good: Named constant
const SecondsPerDay = 24 * 60 * 60
time.Sleep(SecondsPerDay * time.Second)
```

### 3. Be Careful with Integer Division

```go
result := 5 / 2      // 2 (integer division!)
result := 5.0 / 2    // 2.5 (floating-point division)
result := float64(5) / 2  // 2.5 (explicit conversion)
```
