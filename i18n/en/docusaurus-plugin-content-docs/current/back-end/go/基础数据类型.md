# Basic Data Types - Numbers, Strings, Booleans

Go's type system is concise yet powerful. Let's dive deep into the basic data types.

## InputNumbers Numeric Types

### 1. Integer Types

Go provides a rich set of integer types:

```go
// Signed integers
var age int8    // -128 to 127
var level int16 // -32768 to 32767
var score int32 // -2 billion to 2 billion
var money int64 // Very large integers

// Unsigned integers
var count uint8  // 0 to 255
var port uint16  // 0 to 65535
var size uint32  // 0 to 4 billion
var big uint64   // Very large positive numbers

// Platform-dependent
var num int   // 32-bit on 32-bit systems, 64-bit on 64-bit systems
var size uint // Usually used for indices and counts
```

**Practical advice:**
- Default to `int` and `uint`
- Use `int32`, `int64` only when specific size is needed
- Use explicit-sized types for bit operations

### 2. Floating Point Types

```go
var price float32  // Single precision, 6-7 significant digits
var pi float64     // Double precision, 15-16 significant digits

// Recommendation: default to float64
price := 99.99
pi := 3.141592653589793
```

**Note:**
- `float32` has lower precision, use with caution
- For financial calculations, use `decimal` library (to avoid precision issues)

### 3. Complex Number Types

```go
var c1 complex64  // float32 real + float32 imaginary
var c2 complex128 // float64 real + float64 imaginary

// Create complex number
c := complex(3, 4)  // 3 + 4i

// Built-in functions
realPart := real(c)  // 3.0 - get real part
imagPart := imag(c)  // 4.0 - get imaginary part
```

## BarChart Numeric Operations

### Basic Operations

```go
a, b := 10, 3

// Arithmetic operations
sum := a + b        // 13
diff := a - b       // 7
prod := a * b       // 30
quot := a / b       // 3 (integer division!)
rem := a % b        // 1 (modulo)
```

### Bitwise Operations

```go
a, b := 5, 3  // Binary: 101, 011

and := a & b   // 1 (001) - AND
or := a | b    // 7 (111) - OR
xor := a ^ b   // 6 (110) - XOR
not := ^a      // -6 (bitwise NOT)

// Shift operations
left := a << 2   // 20 (10100) - left shift
right := a >> 1  // 2 (10) - right shift
```

### Comparison Operations

```go
a, b := 10, 20

eq := a == b   // false - equal
ne := a != b   // true - not equal
lt := a < b    // true - less than
le := a <= b   // true - less than or equal
gt := a > b    // false - greater than
ge := a >= b   // false - greater than or equal
```

### Assignment Operations

```go
a := 10

a += 5   // a = a + 5 = 15
a -= 3   // a = a - 3 = 12
a *= 2   // a = a * 2 = 24
a /= 4   // a = a / 4 = 6
a %= 4   // a = a % 4 = 2

// Bitwise assignment
a &= 3   // AND assignment
a |= 3   // OR assignment
a ^= 3   // XOR assignment
a <<= 2  // Left shift assignment
a >>= 1  // Right shift assignment
```

## Abc String Types

Strings are immutable sequences of bytes in Go.

### Creating Strings

```go
// Double quotes - interpreted string
name := "Hello World"
path := "C:\\Users\\John"  // Backslash needs escaping

// Backticks - raw string literal
multiline := `
This is a
multiline string
no need to escape \ or "
`
```

### String Operations

```go
s := "Hello, Go!"

// Length
length := len(s)  // 11 bytes (not characters!)

// Concatenation
greeting := "Hello" + " " + "World"  // "Hello World"

// Indexing (returns byte, not rune)
first := s[0]  // 72 ('H')

// Slicing
sub := s[0:5]  // "Hello"

// Immutability
// s[0] = 'h'  // Compile error: cannot assign to s[0]
```

### Unicode and Runes

```go
// Chinese string
chinese := "你好，世界"

// Wrong way: len() returns bytes
byteLen := len(chinese)  // 15 bytes (UTF-8 encoded)

// Correct way: count runes (Unicode code points)
runeCount := utf8.RuneCountInString(chinese)  // 5 characters

// Iterate over runes
for _, r := range chinese {
    fmt.Printf("%c ", r)  // 你 好 ， 世 界
}
```

### Common String Functions

```go
import "strings"

s := "Hello, World!"

// Search
strings.Contains(s, "World")     // true
strings.HasPrefix(s, "Hello")    // true
strings.HasSuffix(s, "!")        // true
strings.Index(s, "o")            // 4 (first occurrence)
strings.LastIndex(s, "o")        // 8 (last occurrence)

// Transform
strings.ToUpper(s)       // "HELLO, WORLD!"
strings.ToLower(s)       // "hello, world!"
strings.Title(s)         // "Hello, World!"

// Manipulate
strings.Replace(s, "World", "Go", 1)  // "Hello, Go!"
strings.Split(s, ", ")               // ["Hello", "World!"]
strings.Join([]string{"Hello", "Go"}, " ")  // "Hello Go"
strings.TrimSpace("  hello  ")        // "hello"

// Check
strings.ContainsAny(s, "aeiou")  // true (contains vowels)
```

### String Formatting

```go
import "fmt"

name := "John"
age := 25
pi := 3.14159

// Printf formatting
fmt.Printf("Name: %s\n", name)           // String
fmt.Printf("Age: %d\n", age)             // Integer
fmt.Printf("Pi: %f\n", pi)               // Float (default 6 decimals)
fmt.Printf("Pi: %.2f\n", pi)             // Float (2 decimals)
fmt.Printf("Binary: %b\n", age)          // Binary
fmt.Printf("Hex: %x\n", age)             // Hexadecimal
fmt.Printf("Scientific: %e\n", pi)       // Scientific notation
fmt.Printf("Type: %T\n", age)            // Type of variable
fmt.Printf("Value: %v\n", age)           // Default format
fmt.Printf("Quote: %q\n", name)          // Quoted string
fmt.Printf("Pointer: %p\n", &age)        // Pointer address

// Sprintf returns string instead of printing
result := fmt.Sprintf("%s is %d years old", name, age)
```

## WhiteCheckMark Boolean Type

Booleans represent true or false values.

### Declaration and Operations

```go
var isActive bool = true
isReady := false

// Logical operations
and := isActive && isReady   // false (AND)
or := isActive || isReady    // true (OR)
not := !isActive             // false (NOT)
```

### Comparison Results

```go
a, b := 10, 20

greater := a > b      // false
less := a < b         // true
equal := a == b       // false
notEqual := a != b    // true
```

### Boolean Expressions

```go
age := 25
hasID := true

// Combined conditions
canEnter := age >= 18 && hasID  // true
isVIP := age >= 60 || age < 12  // false

// Short-circuit evaluation
// If first condition is false, second won't be evaluated
result := false && expensiveFunction()  // expensiveFunction won't run
```

## ArrowsClockwise Type Conversion

Go requires explicit type conversion (no implicit conversion).

### Numeric Conversion

```go
var i int = 42
var f float64 = float64(i)    // int to float64
var u uint = uint(i)          // int to uint

// Precision loss possible
bigFloat := 3.99
smallInt := int(bigFloat)     // 3 (truncated, not rounded)
```

### String Conversion

```go
import "strconv"

// Number to string
num := 42
str := strconv.Itoa(num)           // "42"
str = fmt.Sprintf("%d", num)       // "42"

// String to number
s := "42"
n, err := strconv.Atoi(s)          // 42, nil
if err != nil {
    // Handle error
}

// Parse with base
n, _ = strconv.ParseInt("1010", 2, 64)   // 10 (binary)
n, _ = strconv.ParseInt("FF", 16, 64)    // 255 (hex)

// Float parsing
f, _ := strconv.ParseFloat("3.14", 64)
```

### Byte and Rune Conversion

```go
// String to bytes
s := "Hello"
bytes := []byte(s)     // [72 101 108 108 111]

// Bytes to string
original := string(bytes)  // "Hello"

// String to runes
runes := []rune("你好")   // [20320 22909]

// Rune to string
r := 'A'
str := string(r)       // "A"
```

## LightBulb Best Practices

### 1. Choose Appropriate Types

```go
// WhiteCheckMark Good choices
var count int           // General counting
var price float64       // Monetary values (or use decimal library)
var name string         // Text data
var isValid bool        // Flags

// CrossMark Poor choices
var flag int            // Use bool instead
var tinyNumber int64    // Overkill for small ranges
```

### 2. Be Careful with Integer Division

```go
a, b := 5, 2

// CrossMark Unexpected result
result := a / b  // 2, not 2.5!

// WhiteCheckMark Correct approach
result := float64(a) / float64(b)  // 2.5
```

### 3. Handle String Encoding

```go
// CrossMark Wrong for Unicode
s := "你好"
length := len(s)  // 6 bytes, not 2 characters!

// WhiteCheckMark Correct
import "unicode/utf8"
length := utf8.RuneCountInString(s)  // 2 characters
```

### 4. Check Conversion Errors

```go
// CrossMark Ignoring errors
num, _ := strconv.Atoi(userInput)

// WhiteCheckMark Proper error handling
num, err := strconv.Atoi(userInput)
if err != nil {
    log.Printf("Invalid input: %v", err)
    return
}
```

## WhiteCheckMark Key Points Summary

- WhiteCheckMark Go has rich numeric types (signed/unsigned, various sizes)
- WhiteCheckMark Default to `int` and `float64` unless specific size needed
- WhiteCheckMark Strings are immutable byte sequences
- WhiteCheckMark Use `rune` for Unicode characters
- WhiteCheckMark Booleans support `&&`, `||`, `!` operations
- WhiteCheckMark Type conversions must be explicit
- WhiteCheckMark Check errors when converting strings to numbers
