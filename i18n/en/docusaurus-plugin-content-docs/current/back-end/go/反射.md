# Reflection - Inspecting Types at Runtime

Reflection allows programs to inspect and modify their own structure.

## 🔍 Basic Usage

```go
import "reflect"

type Person struct {
    Name string
    Age  int
}

p := Person{Name: "John", Age: 25}
t := reflect.TypeOf(p)
v := reflect.ValueOf(p)

fmt.Println(t.Name())        // Person
fmt.Println(v.Field(0))      // John
```
## 🎯 Common Use Cases

### Struct Tags

```go
type User struct {
    Name  string `json:"name"`
    Email string `json:"email,omitempty"`
}

// Parse tags at runtime
t := reflect.TypeOf(User{})
field, _ := t.FieldByName("Name")
tag := field.Tag.Get("json")  // "name"
```

### Dynamic Method Call

```go
v := reflect.ValueOf(obj)
method := v.MethodByName("MethodName")
args := []reflect.Value{reflect.ValueOf(arg)}
method.Call(args)
```

## ⚠️ Performance Consideration

Reflection is powerful but slow. Avoid in hot paths.
