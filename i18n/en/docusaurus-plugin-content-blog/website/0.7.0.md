## 🌍 Version 0.7.0: Smart Language Detection

### Feature Overview

New intelligent language detection system based on browser language and geolocation, providing friendly language switching prompts for non-Chinese users.

**Core Features:**

- Automatic browser `navigator.language` detection
- Fallback IP geolocation detection
- Bidirectional switching prompts (Chinese/English)
- Glassmorphism floating UI
- 24-hour fatigue control
- Full platform support (PC/Mobile)

---

### 🎯 Detection Logic

#### Priority Strategy

| Priority | Detection Method | Description |
|----------|-----------------|-------------|
| 1 | Browser Language | `navigator.language` / `navigator.userLanguage` |
| 2 | IP Geolocation | Call `ipapi.co` API to determine country |
| 3 | Default Behavior | Stay on current language if detection fails |

#### Trigger Conditions

- **Chinese site visitor**: System language is English/non-Chinese → Prompt switch to English
- **English site visitor**: System language is Chinese/IP in CN → Prompt switch to Chinese

### 💡 User Experience Design

#### Bottom Floating Prompt

```
┌─────────────────────────────────────┐
│  Switch to English?             [×] │
│                                     │
│  [   Stay   ] [ Switch ]            │
└─────────────────────────────────────┘
```

**Design Highlights:**

- Triggered using DOMContentLoaded (faster response)
- Auto-dismiss after 15 seconds
- Glassmorphism visual effect
- Light/dark mode adaptive

#### Fatigue Mechanism

| User Action | Effect |
|-------------|--------|
| Click "Switch" | No prompt for 24 hours |
| Click "Stay" | No prompt for 24 hours |
| Click "×" Close | No prompt for 24 hours |
| Manual language change | No prompt for 24 hours |

**Implementation:** Store user preference in `localStorage`, reset after 24 hours.

#### Key Code

**Detection Logic:**

```typescript
// Browser language detection
export function detectBrowserLanguage(): DetectionResult | null {
  const lang = navigator.language || navigator.userLanguage;

  if (isChineseLanguage(lang)) {
    return { shouldRedirect: false, targetLocale: 'zh-CN', ... };
  }

  if (isEnglishLanguage(lang)) {
    return { shouldRedirect: true, targetLocale: 'en', ... };
  }

  // Other languages default to English
  return { shouldRedirect: true, targetLocale: 'en', ... };
}
```

**Component Integration:**

```tsx

    </>
  );
}
```

### 📈 Performance Optimization

| User Type | Network Requests | Note |
|-----------|-----------------|------|
| Chinese users | 0 | Browser detection only |
| English users | 0 | Browser detection only |
| Other languages | 0 | Browser detection, direct use |
| Browser detection failed | 1 | Rare case, call ipapi.co |

**Key Improvements:**
- DOMContentLoaded trigger instead of fixed delay
- Browser detection prioritized, IP as fallback only
- Minimal network requests for 99%+ users

### 🎉 Summary

This update fills an important gap for internationalized sites—**seamless language guidance**. Through intelligent detection + friendly prompts + fatigue control, non-Chinese users can immediately discover the English version without disturbing Chinese users.

The implementation follows progressive enhancement principles:
- Detection fails? Silently ignore
- Browser unsupported? Degrade to normal alert
- localStorage disabled? Re-detect on next visit

User experience always comes first 🚀
