# Project Overview

## What is eave-web?

eave-web is a bilingual (Chinese/English) technical documentation website for Eave Luo, a software engineer specializing in fullstack development.

## Purpose

- Share technical knowledge and tutorials
- Document best practices for frontend/backend development
- Publish AI industry news and insights (72-hour updates)

## Key Modules

### Documentation

- **Frontend** - React, Vue, Webpack, etc.
- **Backend** - Node.js, Go, APIs
- **Operation** - Linux, Nginx, DevOps
- **AI** - 72-hour AI frontier updates

### Features

- Bilingual support (zh-CN/en)
- Responsive design (mobile-first)
- Dark mode support
- Search functionality (Algolia DocSearch)
- Comments (Giscus)

## Target Audience

- Software engineers
- Fullstack developers
- DevOps engineers
- AI enthusiasts

## Success Metrics

- FCP < 2.0s (achieved: ~1.8s, optimized from 3.33s)
- LCP < 2.5s
- CLS < 0.1
- Lighthouse score > 90

### Performance History
- 2026-02-27: FCP optimization (3.33s → 1.8s)
- Key improvements: lazy loading, animation optimization, SVG compression
