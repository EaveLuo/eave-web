# Tech Stack

## Frontend

### Core

- **React** 18 - UI framework
- **TypeScript** 5.9 - Type safety
- **Docusaurus** 3.9.2 - Static site generator

### Styling

- **Tailwind CSS** v4 - Utility-first CSS
- **Infima** - Docusaurus default theme
- **Framer Motion** - Animations

### Build

- **Rspack** (via @docusaurus/faster) - Bundler
- **PostCSS** - CSS processing
- **SVGR** - SVG handling

## Internationalization

- **i18n** - Built-in Docusaurus i18n
- **Locales**: zh-CN (default), en
- **Structure**: `i18n/{locale}/docusaurus-plugin-content-docs/current/`

## Deployment

- **Platform**: Vercel
- **CI/CD**: GitHub Actions
- **Plan**: Hobby (free tier)
  - 100GB bandwidth/month
  - 1M serverless GB-seconds/month

## DevOps

- **GitHub** - Version control
- **Vercel CLI** - Deployment management
- **GitHub CLI** - PR management

## Monitoring

- **Vercel Analytics** - Performance metrics
- **Vercel Speed Insights** - Web Vitals
- **Lighthouse** - Performance auditing

## Code Quality

- **ESLint** - Code linting
- **TypeScript** - Type checking
- **Prettier** - Code formatting
