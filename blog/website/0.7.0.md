## 🌍 0.7.0 版本：智能语言检测

### 功能概述

新增基于浏览器语言和地理位置的智能语言检测系统，为非中文环境用户提供友好的语言切换提示。

**核心特性：**

- 自动检测浏览器 `navigator.language`
- 备用 IP 地理位置检测
- 双向切换提示（中文/英文互切）
- 24 小时疲劳度控制
- PC/移动端全平台支持

---

### 🎯 检测逻辑

#### 优先级策略

| 优先级 | 检测方式 | 说明 |
|--------|----------|------|
| 1 | 浏览器语言 | `navigator.language` / `navigator.userLanguage` |
| 2 | IP 地理位置 | 调用 `ipapi.co` API 判断国家 |
| 3 | 默认行为 | 无法检测时保持当前语言 |

#### 触发条件

- **中文站访问者**：系统语言为英文/非中文 → 提示切换英文
- **英文站访问者**：系统语言为中文/IP 位于 CN → 提示切换中文

### 💡 用户体验设计

#### 底部浮层提示

```
┌─────────────────────────────────────┐
│  Switch to English?             [×] │
│                                     │
│  [   Stay   ] [ Switch ]            │
└─────────────────────────────────────┘
```

**设计要点：**

- 页面加载后 2 秒异步触发（避免影响首屏）
- 15 秒后自动消失
- 毛玻璃视觉效果
- 支持深浅色模式自适应

#### 疲劳度机制

| 用户行为 | 效果 |
|----------|------|
| 点击「Switch」切换 | 24 小时内不再提示 |
| 点击「Stay」保持 | 24 小时内不再提示 |
| 点击「×」关闭 | 24 小时内不再提示 |
| 手动切换语言 | 24 小时内不再提示 |

**技术实现：**

使用 `localStorage` 存储用户偏好，24 小时后自动重置，第二天再次访问时重新检测。

#### 核心代码

**检测逻辑：**

```typescript
// 浏览器语言检测
export function detectBrowserLanguage(): DetectionResult | null {
  const lang = navigator.language || navigator.userLanguage;

  const chineseCodes = ['zh', 'zh-CN', 'zh-TW', 'zh-HK', 'cmn-Hans'];
  const englishCodes = ['en', 'en-US', 'en-GB', 'en-CA', 'en-AU'];

  if (chineseCodes.some(code => lang.startsWith(code))) {
    return { shouldRedirect: false, targetLocale: 'zh-CN', ... };
  }

  if (englishCodes.some(code => lang.startsWith(code))) {
    return { shouldRedirect: true, targetLocale: 'en', ... };
  }

  // 其他语言默认英文
  return { shouldRedirect: true, targetLocale: 'en', ... };
}

// IP 地理位置检测
export async function detectByGeoIP(): Promise<DetectionResult | null> {
  const response = await fetch('https://ipapi.co/json/');
  const data = await response.json();
  const isChina = data.country_code === 'CN';

  return {
    shouldRedirect: !isChina,
    targetLocale: isChina ? 'zh-CN' : 'en',
    ...
  };
}
```

#### 国际化文案

支持中英文提示文案：

| Key | 英文 | 中文 |
|-----|------|------|
| `languagePrompt.switchToEn` | Switch to English? | 切换到 English？ |
| `languagePrompt.switchToZh` | Switch to Chinese? | 切换到中文？ |
| `languagePrompt.stay` | Stay | 保持现状 |
| `languagePrompt.switch` | Switch | 切换 |
| `languagePrompt.close` | Close | 关闭 |

### 🎉 总结

这次更新为国际化站点补上了重要的一环——**无感的语言引导**。通过智能检测 + 友好提示 + 疲劳度控制，让非中文用户第一时间发现英文版本，同时又不打扰中文用户。

整个实现遵循了渐进增强原则：
- 检测失败？静默忽略
- 浏览器不支持？降级为普通弹窗
- localStorage 禁用？下次访问重新检测

用户体验永远是第一位 🚀
