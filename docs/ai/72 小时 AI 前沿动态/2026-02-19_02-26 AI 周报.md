## 📰 行业热点

### 11. Google API 密钥安全漏洞（Hacker News 512 分）

**来源：** [Truffle Security](https://trufflesecurity.com/blog/google-api-keys-werent-secrets-but-then-gemini-changed-the-rules)

**问题：** Gemini API 启用后，原本公开的 Google Maps API 密钥可访问敏感端点。

**影响：**
- 发现 2,863 个可访问 Gemini 的 API 密钥
- 包括 Google 自身密钥（最早部署于 2023 年）
- 可能导致未授权访问和账单费用

**建议：** 立即检查并轮换 API 密钥，分离不同服务的密钥。
