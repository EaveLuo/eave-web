### 4. Google API 密钥安全风险：Gemini 导致权限升级漏洞

**时间：** 2026-02-26  
**来源：** [Simon Willison's Weblog](https://simonwillison.net/) / [Truffle Security](https://trufflesecurity.com/)

安全研究人员发现 **Google API 密钥**存在严重安全隐患：

**问题根源：**
- Google Maps API 密钥设计为公开（嵌入网页）
- Gemini API 启用后，相同密钥可访问敏感端点
- 开发者未收到密钥权限变更警告

**影响范围：**
- Truffle Security 在 2025 年 11 月的 Common Crawl 中发现 **2,863 个**可访问 Gemini 的 API 密钥
- 包括 Google 自身的密钥（最早部署于 2023 年 2 月）

**建议：** 立即检查并轮换可能受影响的 API 密钥。

### Claude Code 推出远程控制功能

**时间：** 2026-02-25  
**来源：** [Anthropic](https://code.claude.com/docs/en/remote-control)

Claude Code 新增 **远程控制** 功能：

**功能特性：**
- 通过 Web/iOS/桌面端向本地计算机发送指令
- 支持单会话控制
- 需手动批准每个操作（暂不支持 --dangerously-skip-permissions）

**当前限制：**
- 仅支持单会话
- 需要本地计算机保持唤醒
- 偶发 API 500 错误

**对比：** 与 OpenClaw 等工具相比，Claude Code 仍缺少定时任务功能。
