# 结构体 - 面向对象的新思路

Go 没有类，但用结构体实现了类似的封装效果。让我们探索 Go 式的"面向对象"。

## 📦 定义结构体

```go
type Person struct {
    Name string
    Age  int
    City string
}
```

## 🎯 创建实例

```go
// 方式 1：字面量
p1 := Person{Name: "张三", Age: 25, City: "北京"}

// 方式 2：部分字段（其他为零值）
p2 := Person{Name: "李四"}  // Age=0, City=""

// 方式 3：按顺序（不推荐）
p3 := Person{"王五", 30, "上海"}

// 方式 4：new 创建（返回指针）
p4 := new(Person)
p4.Name = "赵六"
```

## 🔧 访问和修改

```go
p := Person{Name: "张三", Age: 25}

// 访问字段
fmt.Println(p.Name)  // 张三

// 修改字段
p.Age = 26

// 指针访问（自动解引用）
pp := &p
fmt.Println(pp.Name)  // 张三（等价于 (*pp).Name）
pp.City = "深圳"
```

## 🏷️ 结构体标签

```go
type User struct {
    ID       int    `json:"id"`
    Name     string `json:"name"`
    Password string `json:"-"`              // 忽略
    Email    string `json:"email,omitempty"` // 空时省略
}
```

**常用场景：**
- JSON 序列化
- 表单解析
- 数据库映射

## 🔀 匿名字段（嵌入）

```go
type Base struct {
    ID   int
    Name string
}

type User struct {
    Base  // 匿名字段
    Email string
}

// 使用
u := User{
    Base: Base{ID: 1, Name: "张三"},
    Email: "zhang@example.com",
}

// 可以直接访问嵌入的字段
fmt.Println(u.ID)    // 1（等价于 u.Base.ID）
fmt.Println(u.Name)  // 张三
```

## 🎨 组合优于继承

```go
// ❌ Go 没有继承
// type Admin extends Person { ... }

// ✅ Go 用组合
type Person struct {
    Name string
    Age  int
}

type Admin struct {
    Person      // 嵌入 Person
    Permissions []string
}

// Admin 自动拥有 Person 的所有字段
admin := Admin{
    Person: Person{Name: "张三", Age: 25},
    Permissions: []string{"read", "write"},
}

fmt.Println(admin.Name)  // 张三
```

## 📊 结构体比较

```go
p1 := Person{Name: "张三", Age: 25}
p2 := Person{Name: "张三", Age: 25}
p3 := Person{Name: "李四", Age: 25}

// 结构体可以直接比较（所有字段都可比较时）
fmt.Println(p1 == p2)  // true
fmt.Println(p1 == p3)  // false

// 包含切片的结构体不能比较
type Invalid struct {
    Name   string
    Tags   []string  // ❌ 不能比较
}
```

## 💡 构造函数模式

```go
// 构造函数（约定俗成用 New 开头）
func NewPerson(name string, age int) *Person {
    return &Person{
        Name: name,
        Age:  age,
    }
}

// 使用选项模式（更灵活）
type PersonOption func(*Person)

func WithCity(city string) PersonOption {
    return func(p *Person) {
        p.City = city
    }
}

func NewPersonWithOptions(name string, age int, opts ...PersonOption) *Person {
    p := &Person{
        Name: name,
        Age:  age,
    }
    for _, opt := range opts {
        opt(p)
    }
    return p
}

// 使用
p := NewPersonWithOptions("张三", 25, WithCity("北京"))
```

## 🐛 常见错误

### 1. 忘记字段名

```go
// ❌ 错误
p := Person{"张三", 25, "北京"}  // 容易搞错顺序

// ✅ 推荐
p := Person{
    Name: "张三",
    Age:  25,
    City: "北京",
}
```

### 2. 混淆值和指针

```go
type Counter struct{ count int }

func (c Counter) Inc() { c.count++ }      // 值接收者
func (c *Counter) Inc2() { c.count++ }    // 指针接收者

c := Counter{}
c.Inc()   // c.count 不变
c.Inc2()  // c.count 变了
```

### 3. 导出字段规则

```go
type Config struct {
    name  string  // 小写=私有
    Name  string  // 大写=导出
    age   int     // 私有
    Age   int     // 导出
}
```

## ✅ 知识点总结

- ✅ 结构体是字段的集合
- ✅ 用字面量或 new 创建
- ✅ 标签用于元数据（JSON 等）
- ✅ 匿名字段实现组合
- ✅ 结构体可以直接比较
- ✅ 大写字段=导出，小写=私有
