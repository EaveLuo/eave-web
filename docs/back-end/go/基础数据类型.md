# 基础数据类型 - 数字、字符串、布尔

Go 的类型系统简洁而强大。让我们深入了解基础数据类型。

## 🔢 数字类型

### 1. 整数类型

Go 提供了丰富的整数类型：

```go
// 有符号整数
var age int8    // -128 到 127
var level int16 // -32768 到 32767
var score int32 // -21 亿到 21 亿
var money int64 // 超大整数

// 无符号整数
var count uint8  // 0 到 255
var port uint16  // 0 到 65535
var size uint32  // 0 到 42 亿
var big uint64   // 超大正数

// 根据平台自动选择
var num int   // 32 位系统=32 位，64 位系统=64 位
var size uint // 通常用于索引和计数
```

**实际建议：**
- 默认使用 `int` 和 `uint`
- 需要特定大小时才用 `int32`、`int64`
- 位运算时用明确大小的类型

### 2. 浮点类型

```go
var price float32  // 单精度，6-7 位有效数字
var pi float64     // 双精度，15-16 位有效数字

// 推荐：默认使用 float64
price := 99.99
pi := 3.141592653589793
```

**注意：**
- `float32` 精度较低，谨慎使用
- 金融计算建议用 `decimal` 库（避免精度问题）

### 3. 复数类型

```go
var c1 complex64  // float32 实部 + float32 虚部
var c2 complex128 // float64 实部 + float64 虚部

// 创建复数
c := complex(3, 4)  // 3 + 4i

// 内置函数
real := real(c)  // 3.0 - 取实部
imag := imag(c)  // 4.0 - 取虚部
```

## 📊 数字运算

### 基本运算

```go
a, b := 10, 3

// 算术运算
sum := a + b        // 13
diff := a - b       // 7
prod := a * b       // 30
quot := a / b       // 3 (整数除法！)
rem := a % b        // 1 (取模)

// 注意：整数除法会截断
fmt.Println(10 / 3)   // 3
fmt.Println(10.0 / 3) // 3.333333
```

### 自增自减

```go
count := 0
count++  // 1
count++  // 2
count--  // 1

// ❌ 错误：Go 没有 ++count 或 count--
// ++count  
// count--
```

**重要：** Go 只有 `count++` 和 `count--`，且只能作为独立语句！

### 位运算

```go
a := 60        // 0011 1100
b := 13        // 0000 1101

// 位运算
and := a & b   // 0000 1100 = 12
or := a | b    // 0011 1101 = 61
xor := a ^ b   // 0011 0001 = 49
not := ^a      // 1100 0011 = -61 (补码)

// 移位
left := a << 2  // 1111 0000 = 240 (左移)
right := a >> 2 // 0000 1111 = 15  (右移)
```

## 🔤 字符串

### 1. 基本用法

```go
// 字符串声明
name := "张三"
message := "Hello, 世界！"

// 多行字符串（反引号）
html := `
<html>
  <body>
    <h1>Hello</h1>
  </body>
</html>
`

// 空字符串
empty := ""
```

### 2. 字符串操作

```go
s := "Hello, Go!"

// 长度
length := len(s)  // 11

// 索引（字节级别）
first := s[0]     // 'H' 的 ASCII 值 72

// 切片
sub := s[0:5]     // "Hello"
sub2 := s[:5]     // "Hello" (省略开始)
sub3 := s[7:]     // "Go!" (省略结束)

// 遍历（注意中文！）
for i := 0; i < len(s); i++ {
    fmt.Printf("%c ", s[i])  // 按字节遍历
}

// 正确遍历 Unicode
for i, ch := range s {
    fmt.Printf("%c ", ch)  // 按字符遍历
}
```

### 3. 字符串拼接

```go
// 方式 1：+ 号（简单场景）
name := "张三"
greeting := "你好，" + name + "！"

// 方式 2：fmt.Sprintf（格式化）
age := 25
info := fmt.Sprintf("%s 今年 %d 岁", name, age)

// 方式 3：strings.Builder（高性能）
var builder strings.Builder
builder.WriteString("Hello")
builder.WriteString(", ")
builder.WriteString("Go!")
result := builder.String()
```

### 4. 常用字符串函数

```go
import "strings"

s := "Hello, Go! Hello, World!"

// 查找
idx := strings.Index(s, "Go")      // 7
cnt := strings.Count(s, "Hello")   // 2
has := strings.Contains(s, "World") // true

// 替换
newS := strings.Replace(s, "Hello", "Hi", 1)
// "Hi, Go! Hello, World!" (只替换 1 个)

// 分割
parts := strings.Split(s, ", ")
// ["Hello", "Go!", "Hello", "World!"]

// 连接
joined := strings.Join(parts, "-")
// "Hello-Go!-Hello-World!"

// 大小写
upper := strings.ToUpper(s)  // 大写
lower := strings.ToLower(s)  // 小写

// 修剪空格
trimmed := strings.TrimSpace("  hello  ")  // "hello"
```

### 5. 字符串不可变

```go
s := "Hello"
// s[0] = 'h'  // ❌ 编译错误！

// 正确方式：创建新字符串
s = "h" + s[1:]  // "hello"
```

## 🔣 字符类型（rune）

Go 中，字符用 `rune` 表示（实际上是 `int32` 的别名）：

```go
// rune 表示 Unicode 码点
var ch rune = '中'
var ch2 rune = 'A'

// 类型转换
ch3 := rune('好')
codePoint := int(ch3)  // Unicode 码点

// 字符串转 rune 切片（正确处理中文）
s := "你好"
runes := []rune(s)
fmt.Println(len(runes))  // 2 (不是字节数！)
```

## ✅ 布尔类型

### 1. 基本用法

```go
var isReady bool = true
var isValid bool = false

// 简短声明
done := false
success := true
```

### 2. 布尔运算

```go
a, b := true, false

// 与、或、非
and := a && b   // false
or := a || b    // true
not := !a       // false

// 短路求值
func check() bool {
    fmt.Println("check called")
    return true
}

if false && check() {
    // check() 不会被调用（短路）
}
```

### 3. 比较运算

```go
a, b := 10, 20

// 比较返回布尔值
eq := a == b   // false
ne := a != b   // true
lt := a < b    // true
le := a <= b   // true
gt := a > b    // false
ge := a >= b   // false
```

## 🔄 类型转换

Go **不支持隐式类型转换**，必须显式转换：

### 1. 数字类型转换

```go
// int 转 float
var i int = 42
var f float64 = float64(i)

// float 转 int（截断小数）
var f2 float64 = 3.14
var i2 int = int(f2)  // 3

// 不同大小整数转换
var i3 int32 = 100
var i4 int64 = int64(i3)
```

### 2. 字符串和数字转换

```go
import "strconv"

// 数字转字符串
num := 42
s := strconv.Itoa(num)      // "42" (int)
s2 := strconv.FormatFloat(3.14, 'f', 2, 64)  // "3.14"

// 字符串转数字
n, err := strconv.Atoi("42")        // 42, nil
f, err := strconv.ParseFloat("3.14", 64)  // 3.14, nil
i, err := strconv.ParseInt("100", 10, 64) // 100, nil
```

### 3. 字符串和字节切片

```go
s := "Hello"

// 字符串转字节切片
b := []byte(s)

// 字节切片转字符串
s2 := string(b)

// rune 切片
runes := []rune("你好")
s3 := string(runes)
```

## 🎯 实战示例

```go
package main

import (
    "fmt"
    "strings"
    "strconv"
)

func main() {
    // 1. 数字运算
    price := 99.99
    quantity := 3
    total := price * float64(quantity)
    fmt.Printf("总价：%.2f\n", total)
    
    // 2. 字符串操作
    name := "张三"
    greeting := fmt.Sprintf("你好，%s！", name)
    fmt.Println(greeting)
    
    // 3. 字符串处理
    email := "  USER@Example.com  "
    email = strings.TrimSpace(email)
    email = strings.ToLower(email)
    fmt.Println(email)  // user@example.com
    
    // 4. 类型转换
    ageStr := "25"
    age, _ := strconv.Atoi(ageStr)
    fmt.Printf("明年你将 %d 岁\n", age+1)
    
    // 5. 布尔运算
    isLoggedIn := true
    hasPermission := false
    canAccess := isLoggedIn && hasPermission
    fmt.Printf("可以访问：%v\n", canAccess)
}
```

## 🐛 常见错误

### 1. 整数除法陷阱

```go
// ❌ 错误
avg := 5 / 2  // 2 (不是 2.5！)

// ✅ 正确
avg := 5.0 / 2  // 2.5
// 或
avg := float64(5) / float64(2)
```

### 2. 字符串索引误区

```go
s := "你好"

// ❌ 错误：按字节索引
fmt.Println(s[0])  // 228 (不是'你'！)

// ✅ 正确：转成 rune
runes := []rune(s)
fmt.Println(runes[0])  // 20320 ('你'的 Unicode)
```

### 3. 忽略转换错误

```go
// ❌ 错误
num, _ := strconv.Atoi("abc")  // 错误被忽略

// ✅ 正确
num, err := strconv.Atoi("abc")
if err != nil {
    fmt.Println("转换失败:", err)
}
```

### 4. 浮点数精度

```go
// ❌ 错误：直接比较浮点数
a, b := 0.1+0.2, 0.3
fmt.Println(a == b)  // false!

// ✅ 正确：比较差值
epsilon := 0.000001
fmt.Println(math.Abs(a-b) < epsilon)  // true
```

## 💡 最佳实践

### 1. 选择合适类型

```go
// ✅ 推荐
var count int           // 一般计数
var age int32           // 明确范围
var money int64         // 金额（分）
var ratio float64       // 比例

// ⚠️ 避免
var count float64       // 计数用 int
var money float64       // 金额用 int64 或 decimal
```

### 2. 字符串拼接

```go
// ✅ 少量拼接
s := "Hello" + ", " + "World"

// ✅ 格式化
s := fmt.Sprintf("%s, %d", name, age)

// ✅ 大量拼接
var builder strings.Builder
for i := 0; i < 100; i++ {
    builder.WriteString(strconv.Itoa(i))
}
result := builder.String()
```

### 3. 类型转换明确

```go
// ✅ 清晰
total := float64(price) * float64(quantity)

// ⚠️ 模糊
total := price * quantity  // 类型不明确
```

## ✅ 知识点总结

- ✅ 整数类型：int、uint、int8/16/32/64
- ✅ 浮点类型：float32、float64（推荐）
- ✅ 字符串不可变，用反引号支持多行
- ✅ rune 处理 Unicode 字符
- ✅ 布尔类型：true/false
- ✅ 类型转换必须显式
- ✅ 字符串和数字用 strconv 转换
