# 函数 - Go 的核心构建块

函数是 Go 程序的基本组成单元。Go 的函数简洁而强大，支持多返回值、可变参数等特性。

## 📦 函数定义

### 1. 基本语法

```go
// 无参数、无返回值
func sayHello() {
    fmt.Println("Hello!")
}

// 有参数、无返回值
func greet(name string) {
    fmt.Printf("Hello, %s!\n", name)
}

// 有参数、有返回值
func add(a int, b int) int {
    return a + b
}

// 多个参数（相同类型可合并）
func addMulti(a, b, c int) int {
    return a + b + c
}
```

### 2. 多返回值 ⭐

Go 的函数可以返回多个值：

```go
// 返回两个值
func divide(a, b int) (int, int) {
    quotient := a / b
    remainder := a % b
    return quotient, remainder
}

// 调用
q, r := divide(10, 3)
fmt.Printf("商：%d, 余数：%d\n", q, r)

// 常见模式：返回值 + 错误
func getUser(id int) (User, error) {
    // ...
}

user, err := getUser(1)
if err != nil {
    // 处理错误
}
```

### 3. 命名返回值

返回值可以命名，自动作为变量处理：

```go
func getPerson() (name string, age int) {
    name = "张三"
    age = 25
    return  // 自动返回 name 和 age
}

// 等价于
func getPerson2() (string, int) {
    name := "张三"
    age := 25
    return name, age
}
```

**优点：** 代码更清晰，特别是返回值较多时。

**注意：** 不要过度使用，简单函数直接返回即可。

## 🔁 可变参数

函数可以接受不定数量的参数：

```go
// 可变参数必须是最后一个参数
func sum(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

// 调用
fmt.Println(sum(1, 2, 3))       // 6
fmt.Println(sum(1, 2, 3, 4, 5)) // 15

// 传递切片
nums := []int{1, 2, 3, 4}
fmt.Println(sum(nums...))  // 展开切片
```

**实际应用：**

```go
// 格式化字符串
fmt.Printf("Name: %s, Age: %d", name, age)

// 日志
log.Println("Error occurred:", err, "at", time.Now())
```

## 🎯 函数作为值

在 Go 中，函数是一等公民：

```go
// 定义函数类型
type Operation func(int, int) int

// 函数赋值
var add Operation = func(a, b int) int {
    return a + b
}

// 作为参数传递
func calculate(a, b int, op Operation) int {
    return op(a, b)
}

result := calculate(10, 5, add)
fmt.Println(result)  // 15

// 作为返回值
func getOperation(op string) Operation {
    switch op {
    case "add":
        return func(a, b int) int { return a + b }
    case "mul":
        return func(a, b int) int { return a * b }
    default:
        return nil
    }
}
```

## 🔒 匿名函数和闭包

### 1. 匿名函数

没有名字的函数：

```go
// 定义并立即执行
func() {
    fmt.Println("Hello!")
}()

// 带参数
func(name string) {
    fmt.Printf("Hello, %s!\n", name)
}("张三")

// 赋值给变量
add := func(a, b int) int {
    return a + b
}
fmt.Println(add(2, 3))  // 5
```

### 2. 闭包 ⭐

闭包可以捕获外部变量：

```go
func adder() func(int) int {
    sum := 0  // 外部变量
    return func(x int) int {
        sum += x
        return sum
    }
}

// 使用
add := adder()
fmt.Println(add(1))  // 1
fmt.Println(add(2))  // 3
fmt.Println(add(3))  // 6

// 每次调用都累加
```

**实际应用：**

```go
// 计数器
func counter() func() int {
    count := 0
    return func() int {
        count++
        return count
    }
}

// 缓存
func memoize(f func(int) int) func(int) int {
    cache := make(map[int]int)
    return func(n int) int {
        if result, ok := cache[n]; ok {
            return result
        }
        result := f(n)
        cache[n] = result
        return result
    }
}
```

## ⏰ defer 延迟执行

`defer` 语句延迟到函数返回前执行：

```go
func readFile() {
    file, _ := os.Open("file.txt")
    defer file.Close()  // 函数返回前关闭
    
    // 读取文件...
    // 无论是否发生错误，file.Close() 都会执行
}
```

### 1. 多个 defer

```go
func process() {
    defer fmt.Println("第一步清理")
    defer fmt.Println("第二步清理")
    defer fmt.Println("第三步清理")
    
    fmt.Println("处理中...")
}

// 输出：
// 处理中...
// 第三步清理
// 第二步清理
// 第一步清理
```

**特点：** 后进先出（LIFO），像栈一样。

### 2. defer 求值时机

```go
func test() {
    i := 1
    defer fmt.Println(i)  // 立即求值，输出 1
    
    i = 2
    return
}

// 输出：1
```

**重要：** defer 的参数在声明时就求值，不是执行时！

### 3. 实际应用场景

```go
// 1. 资源清理
func processFile() {
    f, _ := os.Open("file.txt")
    defer f.Close()
    
    // 处理文件...
}

// 2. 解锁
func process() {
    mu.Lock()
    defer mu.Unlock()
    
    // 处理逻辑...
}

// 3. 错误恢复
func safeCall() (err error) {
    defer func() {
        if r := recover(); r != nil {
            err = fmt.Errorf("panic: %v", r)
        }
    }()
    
    // 可能 panic 的代码
}

// 4. 性能计时
func slowOperation() {
    start := time.Now()
    defer func() {
        fmt.Printf("耗时：%v\n", time.Since(start))
    }()
    
    // 耗时操作...
}
```

## 🎯 实战示例

### 1. 工具函数

```go
// 字符串处理
func reverse(s string) string {
    runes := []rune(s)
    for i, j := 0, len(runes)-1; i < j; i, j = i+1, j-1 {
        runes[i], runes[j] = runes[j], runes[i]
    }
    return string(runes)
}

// 数组去重
func unique(nums []int) []int {
    seen := make(map[int]bool)
    result := []int{}
    
    for _, num := range nums {
        if !seen[num] {
            seen[num] = true
            result = append(result, num)
        }
    }
    return result
}
```

### 2. 函数式编程

```go
// Map
func Map(nums []int, f func(int) int) []int {
    result := make([]int, len(nums))
    for i, num := range nums {
        result[i] = f(num)
    }
    return result
}

// Filter
func Filter(nums []int, f func(int) bool) []int {
    result := []int{}
    for _, num := range nums {
        if f(num) {
            result = append(result, num)
        }
    }
    return result
}

// 使用
nums := []int{1, 2, 3, 4, 5}
squared := Map(nums, func(x int) int { return x * x })
evens := Filter(nums, func(x int) bool { return x%2 == 0 })
```

### 3. 错误处理包装器

```go
func withRetry(f func() error, maxRetries int) error {
    var err error
    for i := 0; i < maxRetries; i++ {
        if err = f(); err == nil {
            return nil
        }
        time.Sleep(time.Second)
    }
    return err
}

// 使用
err := withRetry(func() error {
    return connectToDatabase()
}, 3)
```

## 🐛 常见错误

### 1. 忽略错误返回值

```go
// ❌ 错误
result, _ := riskyOperation()

// ✅ 正确
result, err := riskyOperation()
if err != nil {
    // 处理错误
}
```

### 2. defer 在循环中

```go
// ❌ 错误：所有文件最后才关闭
for _, file := range files {
    f, _ := os.Open(file)
    defer f.Close()
}

// ✅ 正确
for _, file := range files {
    func() {
        f, _ := os.Open(file)
        defer f.Close()
        // 处理文件...
    }()
}
```

### 3. 闭包变量捕获

```go
// ❌ 错误
for i := 0; i < 3; i++ {
    go func() {
        fmt.Println(i)  // 可能输出 3,3,3
    }()
}

// ✅ 正确
for i := 0; i < 3; i++ {
    i := i  // 创建新变量
    go func() {
        fmt.Println(i)  // 输出 0,1,2
    }()
}

// 或作为参数传递
for i := 0; i < 3; i++ {
    go func(i int) {
        fmt.Println(i)
    }(i)
}
```

### 4. 返回值被命名但未使用

```go
// ⚠️ 不推荐
func getValues() (a, b, c int) {
    a = 1
    b = 2
    // c 未赋值，返回 0
    return
}
```

## 💡 最佳实践

### 1. 函数简短

```go
// ❌ 函数过长
func process() {
    // 100 行代码...
}

// ✅ 拆分成小函数
func process() {
    validate()
    transform()
    save()
}
```

### 2. 错误处理

```go
// ✅ 清晰的错误处理
result, err := doSomething()
if err != nil {
    return fmt.Errorf("do something: %w", err)
}
```

### 3. 函数命名

```go
// ✅ 动词开头
func GetUser()
func CalculateTotal()
func ValidateInput()

// ✅ 布尔函数用 Is/Has/Can
func IsValid()
func HasPermission()
func CanEdit()
```

### 4. 参数顺序

```go
// ✅ 具体到一般
func copyFile(src, dst string)
func connect(host string, port int)

// ✅ 相似类型放一起
func drawCircle(x, y, radius int)
```
