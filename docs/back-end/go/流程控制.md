# 流程控制 - if、for、switch 的 Go 式写法

Go 的流程控制简洁而强大，没有花哨的语法，只有实用的工具。

## 🔀 if-else 条件判断

### 1. 基本用法

```go
age := 18

if age >= 18 {
    fmt.Println("成年人")
} else {
    fmt.Println("未成年人")
}
```

### 2. 带初始化语句的 if ⭐

这是 Go 的特色语法：

```go
// 在条件判断前执行初始化
if age := getAge(); age >= 18 {
    fmt.Println("成年人")
} else {
    fmt.Println("未成年人")
}

// age 变量只在这个 if 语句块内有效
// fmt.Println(age)  // ❌ 编译错误
```

**优点：** 变量作用域清晰，避免污染外部环境。

### 3. 多条件判断

```go
score := 85

if score >= 90 {
    fmt.Println("优秀")
} else if score >= 80 {
    fmt.Println("良好")
} else if score >= 60 {
    fmt.Println("及格")
} else {
    fmt.Println("不及格")
}
```

### 4. 没有括号的 if

```go
// ✅ Go 风格
if age >= 18 {
    // ...
}

// ❌ 不需要括号
if (age >= 18) {
    // ...
}
```

## 🔁 for 循环

Go **只有 for 循环**，但功能强大！

### 1. 基本 for 循环

```go
// 标准形式
for i := 0; i < 10; i++ {
    fmt.Println(i)
}
```

### 2. while 风格

```go
// 省略初始化语句和后置语句
i := 0
for i < 10 {
    fmt.Println(i)
    i++
}
```

### 3. 无限循环

```go
// 省略所有条件（死循环）
for {
    // 一直执行
    if done {
        break  // 可以跳出
    }
}
```

### 4. range 遍历 ⭐

遍历数组、切片、Map、字符串：

```go
// 遍历数组/切片
nums := []int{1, 2, 3, 4, 5}

// 获取索引和值
for i, num := range nums {
    fmt.Printf("索引：%d, 值：%d\n", i, num)
}

// 只需要值
for _, num := range nums {
    fmt.Println(num)
}

// 只需要索引
for i := range nums {
    fmt.Println(i)
}

// 遍历字符串（按 rune）
s := "你好"
for i, ch := range s {
    fmt.Printf("索引：%d, 字符：%c\n", i, ch)
}

// 遍历 Map
m := map[string]int{
    "a": 1,
    "b": 2,
}

for key, value := range m {
    fmt.Printf("%s: %d\n", key, value)
}

// 只需要 key
for key := range m {
    fmt.Println(key)
}
```

### 5. 循环控制

```go
// break - 跳出循环
for i := 0; i < 10; i++ {
    if i == 5 {
        break  // 跳出整个循环
    }
    fmt.Println(i)  // 输出 0-4
}

// continue - 跳过本次
for i := 0; i < 10; i++ {
    if i%2 == 0 {
        continue  // 跳过偶数
    }
    fmt.Println(i)  // 输出 1,3,5,7,9
}

// 标签 - 跳出多层循环
outer:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if i == 1 && j == 1 {
            break outer  // 跳出外层循环
        }
        fmt.Printf("i=%d, j=%d\n", i, j)
    }
}
```

## 🔀 switch 语句

### 1. 基本用法

```go
day := 3

switch day {
case 1:
    fmt.Println("星期一")
case 2:
    fmt.Println("星期二")
case 3:
    fmt.Println("星期三")
default:
    fmt.Println("其他")
}
```

### 2. 自动 break

Go 的 switch **自动 break**，不需要手动写：

```go
// ✅ Go 自动 break
switch day {
case 1:
    fmt.Println("一")  // 不会继续执行 case 2
case 2:
    fmt.Println("二")
}

// ⚠️ 如果需要穿透（很少用）
switch day {
case 1:
    fmt.Println("一")
    fallthrough  // 继续执行下一个 case
case 2:
    fmt.Println("二")  // 会执行
}
```

### 3. 多条件匹配

```go
month := 3

switch month {
case 3, 4, 5:
    fmt.Println("春季")
case 6, 7, 8:
    fmt.Println("夏季")
case 9, 10, 11:
    fmt.Println("秋季")
case 12, 1, 2:
    fmt.Println("冬季")
}
```

### 4. 带初始化语句

```go
// 类似 if，可以带初始化语句
switch day := getDay(); day {
case 1:
    fmt.Println("周一")
default:
    fmt.Println("其他")
}
```

### 5. 无条件 switch（替代 if-else）

```go
score := 85

switch {
case score >= 90:
    fmt.Println("优秀")
case score >= 80:
    fmt.Println("良好")
case score >= 60:
    fmt.Println("及格")
default:
    fmt.Println("不及格")
}
```

**适用场景：** 多个复杂条件判断时，比 if-else 更清晰。

## 🎯 实战示例

### 1. 判断闰年

```go
func isLeapYear(year int) bool {
    if year%4 != 0 {
        return false
    } else if year%100 != 0 {
        return true
    } else if year%400 != 0 {
        return false
    } else {
        return true
    }
    
    // 或更简洁的写法
    return year%4 == 0 && (year%100 != 0 || year%400 == 0)
}
```

### 2. 斐波那契数列

```go
func fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    
    a, b := 0, 1
    for i := 2; i <= n; i++ {
        a, b = b, a+b
    }
    return b
}
```

### 3. 查找元素

```go
func findIndex(nums []int, target int) int {
    for i, num := range nums {
        if num == target {
            return i
        }
    }
    return -1  // 未找到
}
```

### 4. 成绩等级

```go
func getGrade(score int) string {
    switch {
    case score >= 90:
        return "A"
    case score >= 80:
        return "B"
    case score >= 70:
        return "C"
    case score >= 60:
        return "D"
    default:
        return "F"
    }
}
```

## 🐛 常见错误

### 1. 忘记花括号

```go
// ❌ 错误
if age >= 18
    fmt.Println("成年人")

// ✅ 正确
if age >= 18 {
    fmt.Println("成年人")
}
```

### 2. for 循环变量作用域

```go
// ❌ 错误：闭包陷阱
for i := 0; i < 3; i++ {
    go func() {
        fmt.Println(i)  // 可能输出 3,3,3
    }()
}

// ✅ 正确
for i := 0; i < 3; i++ {
    i := i  // 创建新变量
    go func() {
        fmt.Println(i)  // 输出 0,1,2
    }()
}

// 或
for i := 0; i < 3; i++ {
    go func(i int) {
        fmt.Println(i)
    }(i)
}
```

### 3. range 返回值误解

```go
nums := []int{1, 2, 3}

// ❌ 错误：range 返回索引和值
for i := range nums {
    fmt.Println(nums[i])  // 正确但冗余
}

// ✅ 推荐
for _, num := range nums {
    fmt.Println(num)
}

// 或
for i, num := range nums {
    fmt.Printf("%d: %d\n", i, num)
}
```

### 4. switch 忘记 default

```go
// ⚠️ 建议加上 default
switch status {
case "active":
    // ...
case "inactive":
    // ...
default:
    log.Printf("未知状态：%s", status)
}
```

## 💡 最佳实践

### 1. if 语句简化

```go
// ❌ 冗余
if isValid == true {
    // ...
}

// ✅ 简洁
if isValid {
    // ...
}

// ❌ 冗余
if isValid == false {
    // ...
}

// ✅ 简洁
if !isValid {
    // ...
}
```

### 2. for 循环选择

```go
// 需要索引和值
for i, item := range items {
    // ...
}

// 只需要值
for _, item := range items {
    // ...
}

// 需要修改元素
for i := range items {
    items[i] = transform(items[i])
}
```

### 3. switch 对齐

```go
// ✅ 清晰
switch status {
case "active":
    handleActive()
case "pending":
    handlePending()
case "closed":
    handleClosed()
default:
    logUnknown()
}
```

### 4. 提前返回

```go
// ❌ 嵌套过深
func process(data []int) error {
    if data != nil {
        if len(data) > 0 {
            if data[0] > 0 {
                // 处理逻辑
            }
        }
    }
    return nil
}

// ✅ 提前返回
func process(data []int) error {
    if data == nil {
        return nil
    }
    if len(data) == 0 {
        return nil
    }
    if data[0] <= 0 {
        return nil
    }
    // 处理逻辑
    return nil
}
```

## ✅ 知识点总结

- ✅ if 可以带初始化语句
- ✅ for 可以替代 while
- ✅ range 遍历数组、Map、字符串
- ✅ switch 自动 break
- ✅ switch 可以多条件匹配
- ✅ 无条件 switch 替代 if-else
- ✅ 标签跳出多层循环
