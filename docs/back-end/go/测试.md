# 测试 - 单元测试、基准测试与模糊测试

Go 内置强大的测试工具，无需第三方库。

## 🧪 测试基础

### 测试文件命名

```
calculator.go        # 源代码
calculator_test.go   # 测试文件
```

### 基本测试结构

```go
package calculator

import "testing"

func TestAdd(t *testing.T) {
    result := Add(2, 3)
    expected := 5
    
    if result != expected {
        t.Errorf("Add(2, 3) = %d; expected %d", result, expected)
    }
}
```

### 运行测试

```bash
go test              # 运行所有测试
go test -v           # 详细输出
go test -cover       # 覆盖率
go test -bench=.     # 基准测试
```

## 📋 表格驱动测试

```go
func TestAdd(t *testing.T) {
    tests := []struct {
        name     string
        a, b     int
        expected int
    }{
        {"正数相加", 2, 3, 5},
        {"负数相加", -2, -3, -5},
        {"零相加", 0, 0, 0},
    }
    
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            result := Add(tt.a, tt.b)
            if result != tt.expected {
                t.Errorf("Add(%d, %d) = %d", tt.a, tt.b, result)
            }
        })
    }
}
```

## ⏱️ 基准测试

```go
func BenchmarkAdd(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Add(2, 3)
    }
}
```
