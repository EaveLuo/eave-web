# 方法 - 为类型赋予行为

方法是带有接收者的函数。Go 的方法让类型有了行为，实现面向对象的效果。

## 📦 方法定义

```go
type Person struct {
    Name string
    Age  int
}

// 方法定义（接收者在函数名之前）
func (p Person) Greet() {
    fmt.Printf("你好，我是%s！\n", p.Name)
}

// 调用
p := Person{Name: "张三", Age: 25}
p.Greet()  // 你好，我是张三！
```

## 🔀 值接收者 vs 指针接收者

### 值接收者

```go
func (p Person) GetAge() int {
    return p.Age  // 读取值
}

func (p Person) SetAge(age int) {
    p.Age = age  // ❌ 只修改副本，原对象不变
}
```

### 指针接收者

```go
func (p *Person) GetAge2() int {
    return p.Age
}

func (p *Person) SetAge2(age int) {
    p.Age = age  // ✅ 修改原对象
}
```

### 选择规则

```go
// 用指针接收者：
- 需要修改接收者
- 接收者是大结构体（避免复制）
- 保持一致性（同一类型的所有方法）

// 用值接收者：
- 不需要修改接收者
- 接收者是小类型（int、string）
- 接收者是切片、Map、Channel
```

## 🎯 方法集

```go
type Counter struct {
    count int
}

func (c Counter) Get() int        { return c.count }
func (c *Counter) Inc()           { c.count++ }
func (c *Counter) Reset()         { c.count = 0 }

// Counter 的方法集：Get
// *Counter 的方法集：Get, Inc, Reset

c := Counter{}
c.Get()     // ✅
c.Inc()     // ✅ Go 自动取地址

pc := &Counter{}
pc.Get()    // ✅
pc.Inc()    // ✅
```

## 🎨 封装

Go 用大小写控制可见性：

```go
type Account struct {
    balance int  // 私有字段
}

// 私有方法
func (a *Account) validate(amount int) bool {
    return amount > 0
}

// 公有方法
func (a *Account) Deposit(amount int) error {
    if !a.validate(amount) {
        return errors.New("金额无效")
    }
    a.balance += amount
    return nil
}

func (a *Account) GetBalance() int {
    return a.balance
}
```

## 💡 实用示例

### 1. 链式调用

```go
type Builder struct {
    result string
}

func (b *Builder) Append(s string) *Builder {
    b.result += s
    return b  // 返回自身
}

func (b *Builder) Build() string {
    return b.result
}

// 链式调用
result := (&Builder{}).Append("Hello").Append(" ").Append("World").Build()
```

### 2. 实现接口

```go
type Stringer interface {
    String() string
}

type Person struct {
    Name string
    Age  int
}

func (p Person) String() string {
    return fmt.Sprintf("%s(%d)", p.Name, p.Age)
}

// Person 实现了 Stringer 接口
```

### 3. 方法作为值

```go
type Handler func(string)

type Server struct{}

func (s *Server) HandleRequest(req string) {
    fmt.Println("处理请求:", req)
}

func (s *Server) GetHandler() Handler {
    return s.HandleRequest
}

// 使用
server := &Server{}
handler := server.GetHandler()
handler("GET /api/users")
```

## 🐛 常见错误

### 1. 接收者类型不匹配

```go
type Counter struct{ count int }

func (c Counter) Inc() { c.count++ }

c := Counter{}
c.Inc()  // ✅ 但 c.count 不变！

// 应该用指针接收者
func (c *Counter) Inc2() { c.count++ }
```

### 2. 混用接收者类型

```go
// ❌ 不一致
func (c Counter) Get() int { ... }
func (c *Counter) Set(v int) { ... }

// ✅ 一致
func (c *Counter) Get() int { ... }
func (c *Counter) Set(v int) { ... }
```

### 3. 在方法中修改切片

```go
type Data struct {
    nums []int
}

// ✅ 切片是引用，可以直接修改
func (d *Data) Append(n int) {
    d.nums = append(d.nums, n)
}
```

## ✅ 知识点总结

- ✅ 方法 = 函数 + 接收者
- ✅ 值接收者不修改原对象
- ✅ 指针接收者修改原对象
- ✅ 大小写控制可见性
- ✅ 方法实现接口
- ✅ 保持一致性
