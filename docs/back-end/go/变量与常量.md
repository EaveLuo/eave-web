# 变量与常量 - Go 的声明之道

在 Go 中，声明变量和常量有多种方式。让我们探索 Go 独特的"声明之道"。

## 📦 变量声明

### 1. 标准声明方式

```go
var age int
age = 25

var name string
name = "张三"

var salary float64
salary = 9999.99
```

**语法：** `var 变量名 类型`

**特点：**
- 类型在变量名**后面**（与 C/Java 相反）
- 未初始化的变量有**零值**
  - `int` → `0`
  - `string` → `""`
  - `bool` → `false`
  - 指针 → `nil`

### 2. 带初始化的声明

```go
var age int = 25
var name string = "张三"
var salary float64 = 9999.99
```

### 3. 类型推断（推荐）✨

```go
var age = 25        // 自动推断为 int
var name = "张三"    // 自动推断为 string
var salary = 9999.99 // 自动推断为 float64
```

Go 编译器足够聪明，能从右侧值推断出类型！

### 4. 简短声明（最常用）⭐

```go
age := 25
name := "张三"
salary := 9999.99
```

**这是 Go 中最常用的声明方式！**

**特点：**
- 简洁明了
- 自动类型推断
- **只能在函数内部使用**

### 5. 批量声明

```go
// 标准方式
var (
    name string
    age  int
    salary float64
)

// 带初始化
var (
    name = "张三"
    age  = 25
    salary = 9999.99
)

// 简短声明（多个变量）
name, age, salary := "张三", 25, 9999.99
```

## 🎯 实战示例

```go
package main

import "fmt"

func main() {
    // 方式 1：标准声明
    var age int
    age = 25
    fmt.Println("年龄:", age)
    
    // 方式 2：带初始化
    var name string = "张三"
    fmt.Println("姓名:", name)
    
    // 方式 3：类型推断
    var city = "北京"
    fmt.Println("城市:", city)
    
    // 方式 4：简短声明（推荐！）
    salary := 9999.99
    fmt.Println("薪资:", salary)
    
    // 方式 5：批量声明
    var (
        level   = 1
        exp     = 0
        isAlive = true
    )
    fmt.Printf("等级：%d, 经验：%d, 存活：%v\n", level, exp, isAlive)
}
```

## 🔄 变量赋值与交换

### 基本赋值

```go
age := 25
age = 26  // 重新赋值
```

### 神奇的交换

```go
a := 10
b := 20

// Go 风格的交换（无需临时变量！）
a, b = b, a

fmt.Println(a, b)  // 输出：20 10
```

**原理：** Go 同时计算右侧所有值，然后赋值给左侧。

## 🔒 常量

常量一旦定义，就不能修改。

### 1. 基本用法

```go
const pi = 3.14159
const maxUsers = 1000
const appName = "MyApp"
```

### 2. 批量声明

```go
const (
    Sunday    = 0
    Monday    = 1
    Tuesday   = 2
    Wednesday = 3
    Thursday  = 4
    Friday    = 5
    Saturday  = 6
)
```

### 3. iota 枚举器 ⭐

`iota` 是 Go 的枚举计数器，从 0 开始自动递增：

```go
const (
    Sunday = iota  // 0
    Monday         // 1
    Tuesday        // 2
    Wednesday      // 3
    Thursday       // 4
    Friday         // 5
    Saturday       // 6
)
```

**更酷的用法：**

```go
const (
    _ = iota             // 忽略 0
    KB = 1 << (10 * iota) // 1 << 10 = 1024
    MB                    // 1 << 20 = 1048576
    GB                    // 1 << 30
    TB                    // 1 << 40
)

fmt.Println(KB)  // 1024
fmt.Println(MB)  // 1048576
fmt.Println(GB)  // 1073741824
```

### 4. 常量表达式

```go
const (
    width  = 1920
    height = 1080
    area   = width * height  // 常量表达式，编译时计算
)
```

## ⚖️ 变量 vs 常量

| 特性 | 变量 | 常量 |
|------|------|------|
| 可修改 | ✅ 是 | ❌ 否 |
| 声明关键字 | `var` / `:=` | `const` |
| 类型推断 | ✅ 支持 | ✅ 支持 |
| 作用域 | 任意 | 任意 |
| 编译时计算 | ❌ 否 | ✅ 是 |

## 🎨 命名规范

### 1. 可见性规则

Go 通过**首字母大小写**控制可见性：

```go
// 大写开头 = 导出（公共）
var Name = "张三"  // 其他包可访问

// 小写开头 = 未导出（私有）
var name = "李四"  // 仅当前包可访问
```

**这是 Go 的独特设计！无需 `public`/`private` 关键字。**

### 2. 命名建议

```go
// ✅ 好的命名
var userName string
var userID int
var isValid bool
var totalAmount float64

// ❌ 不好的命名
var a int           // 太短，无意义
var theName string  // 冗余
var flag bool       // 不够明确
```

### 3. 简短命名场景

```go
// 循环变量
for i := 0; i < 10; i++ {
    // i 足够清晰
}

// 错误返回值
err := doSomething()
if err != nil {
    // err 是约定俗成的命名
}

// 接收者（方法中）
func (u *User) GetName() string {
    // u 是常见命名
}
```

## 🐛 常见错误

### 1. 重复声明

```go
// ❌ 错误
age := 25
age := 26  // 编译错误：no new variables

// ✅ 正确
age := 25
age = 26   // 重新赋值，不用 :=
```

### 2. 未使用变量

```go
func main() {
    age := 25  // ❌ 编译错误：declared and not used
    fmt.Println("Hello")
}
```

**Go 编译器会检查未使用的变量！**

**解决方案：**
```go
// 临时注释
// age := 25

// 或使用空白标识符
_ = age
```

### 3. 常量重新赋值

```go
const pi = 3.14
pi = 3.14159  // ❌ 编译错误：cannot assign to pi
```

## 💡 最佳实践

### 1. 优先使用简短声明

```go
// ✅ 推荐
name := "张三"
age := 25

// ⚠️ 除非需要指定类型
var data []byte = make([]byte, 1024)
```

### 2. 批量声明相关变量

```go
// ✅ 清晰
var (
    host = "localhost"
    port = 8080
    path = "/api"
)

// ❌ 分散
var host = "localhost"
var port = 8080
var path = "/api"
```

### 3. 常量用大写

```go
// ✅ 推荐
const (
    MaxRetry = 3
    Timeout  = 30
)

// ⚠️ 不推荐
const (
    maxRetry = 3
    timeout  = 30
)
```

### 4. 有意义的命名

```go
// ✅ 清晰
daysInWeek := 7
maxConnection := 100

// ❌ 模糊
d := 7
m := 100
```

## 🎯 实战练习

```go
package main

import "fmt"

func main() {
    // 1. 声明个人信息
    name := "张三"
    age := 25
    city := "北京"
    
    // 2. 声明工作信息
    var company string = "科技公司"
    var salary float64 = 15000
    level := 5
    
    // 3. 使用常量
    const taxRate = 0.1
    afterTax := salary * (1 - taxRate)
    
    // 4. 交换值
    a, b := 10, 20
    a, b = b, a
    
    // 5. 输出
    fmt.Printf("%s, %d 岁，住在%s\n", name, age, city)
    fmt.Printf("在%s工作，级别%d，税后工资%.2f\n", 
               company, level, afterTax)
    fmt.Printf("交换后：a=%d, b=%d\n", a, b)
}
```

## ✅ 知识点总结

- ✅ 四种变量声明方式
- ✅ 简短声明 `:=` 最常用
- ✅ 常量用 `const` 声明
- ✅ `iota` 自动生成枚举值
- ✅ 大写开头=导出，小写开头=私有
- ✅ Go 检查未使用变量
