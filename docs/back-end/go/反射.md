# 反射 - 运行时的类型洞察

反射让程序在运行时检查和操作类型。这是 Go 的高级特性，强大但需谨慎使用。

## 🔍 什么是反射？

反射允许程序在运行时：
- 检查变量的类型
- 获取结构体字段
- 调用方法
- 修改值

```go
import "reflect"

var x float64 = 3.4
v := reflect.ValueOf(x)

fmt.Println("类型:", v.Type())  // float64
fmt.Println("种类:", v.Kind())  // float64
fmt.Println("值:", v.Float())   // 3.4
```

## 🎯 核心 API

### 1. reflect.TypeOf

```go
type Person struct {
    Name string
    Age  int
}

p := Person{Name: "张三", Age: 25}

t := reflect.TypeOf(p)
fmt.Println("类型:", t)        // main.Person
fmt.Println("种类:", t.Kind()) // struct

// 遍历字段
for i := 0; i < t.NumField(); i++ {
    field := t.Field(i)
    fmt.Printf("字段：%s, 类型：%s\n", field.Name, field.Type)
}
```

### 2. reflect.ValueOf

```go
p := Person{Name: "张三", Age: 25}
v := reflect.ValueOf(p)

// 访问字段
name := v.Field(0).String()
age := v.Field(1).Int()

fmt.Println(name, age)  // 张三 25
```

### 3. 修改值

```go
p := Person{Name: "张三", Age: 25}
v := reflect.ValueOf(&p).Elem()  // 需要指针才能修改

v.Field(0).SetString("李四")
v.Field(1).SetInt(26)

fmt.Println(p)  // {李四 26}
```

## 🎨 实战应用

### 1. 通用 JSON 解析

```go
func parseJSON(data []byte, v interface{}) error {
    rv := reflect.ValueOf(v)
    
    // 检查是否是指针
    if rv.Kind() != reflect.Ptr {
        return errors.New("必须传入指针")
    }
    
    // 解析逻辑...
    return nil
}
```

### 2. 结构体验证

```go
func validate(v interface{}) error {
    val := reflect.ValueOf(v)
    typ := val.Type()
    
    for i := 0; i < val.NumField(); i++ {
        field := val.Field(i)
        tag := typ.Field(i).Tag.Get("validate")
        
        if tag == "required" && field.IsZero() {
            return fmt.Errorf("%s 不能为空", typ.Field(i).Name)
        }
    }
    
    return nil
}

// 使用
type User struct {
    Name string `validate:"required"`
    Age  int    `validate:"required"`
}

err := validate(User{})  // 错误：Name 不能为空
```

### 3. 通用 Map 转换

```go
func structToMap(v interface{}) map[string]interface{} {
    result := make(map[string]interface{})
    val := reflect.ValueOf(v)
    typ := val.Type()
    
    for i := 0; i < val.NumField(); i++ {
        field := typ.Field(i)
        value := val.Field(i).Interface()
        result[field.Name] = value
    }
    
    return result
}
```

### 4. 动态方法调用

```go
type Calculator struct{}

func (c Calculator) Add(a, b int) int {
    return a + b
}

func callMethod(obj interface{}, methodName string, args ...interface{}) []reflect.Value {
    v := reflect.ValueOf(obj)
    method := v.MethodByName(methodName)
    
    if !method.IsValid() {
        panic("方法不存在")
    }
    
    in := make([]reflect.Value, len(args))
    for i, arg := range args {
        in[i] = reflect.ValueOf(arg)
    }
    
    return method.Call(in)
}

// 使用
calc := Calculator{}
results := callMethod(calc, "Add", 10, 20)
fmt.Println(results[0].Int())  // 30
```

## 🐛 常见错误

### 1. 不能修改的值

```go
x := 42
v := reflect.ValueOf(x)

// ❌ panic
v.SetInt(100)  // 不能修改

// ✅ 正确
v2 := reflect.ValueOf(&x).Elem()
v2.SetInt(100)
```

### 2. 类型断言失败

```go
var x interface{} = 42
v := reflect.ValueOf(x)

// ❌ 错误
// floatVal := v.Float()  // panic

// ✅ 正确
if v.Kind() == reflect.Int {
    intVal := v.Int()
}
```

### 3. 性能问题

```go
// ❌ 反射很慢
for i := 0; i < 1000000; i++ {
    v := reflect.ValueOf(obj)
    v.Field(0).Int()
}

// ✅ 直接访问
for i := 0; i < 1000000; i++ {
    _ = obj.Field0
}
```

## 💡 最佳实践

### 1. 优先使用类型断言

```go
// ❌ 过度使用反射
func getName(v interface{}) string {
    return reflect.ValueOf(v).FieldByName("Name").String()
}

// ✅ 类型断言
func getName2(v interface{}) string {
    if p, ok := v.(Person); ok {
        return p.Name
    }
    return ""
}
```

### 2. 缓存反射结果

```go
var fieldCache = make(map[reflect.Type][]reflect.StructField)

func getFields(t reflect.Type) []reflect.StructField {
    if fields, ok := fieldCache[t]; ok {
        return fields
    }
    
    fields := make([]reflect.StructField, t.NumField())
    for i := 0; i < t.NumField(); i++ {
        fields[i] = t.Field(i)
    }
    fieldCache[t] = fields
    return fields
}
```

### 3. 清晰的错误处理

```go
func safeReflect(v interface{}) (err error) {
    defer func() {
        if r := recover(); r != nil {
            err = fmt.Errorf("反射错误：%v", r)
        }
    }()
    
    // 反射操作
    reflect.ValueOf(v).Elem().Field(0).String()
    
    return nil
}
```

## ✅ 知识点总结

- ✅ TypeOf 获取类型信息
- ✅ ValueOf 获取值
- ✅ 需要指针才能修改值
- ✅ 反射性能较低，谨慎使用
- ✅ 优先使用类型断言
- ✅ 缓存反射结果提升性能
