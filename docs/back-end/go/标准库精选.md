# 标准库精选 - 常用包详解

Go 的标准库被称为"电池包含"（Batteries Included）。

## 📦 核心包分类

### I/O 与格式化

| 包 | 用途 | 示例 |
|----|------|------|
| `fmt` | 格式化 I/O | `fmt.Printf`, `fmt.Sprintf` |
| `io` | I/O 原语 | `io.Reader`, `io.Writer` |
| `bufio` | 缓冲 I/O | `bufio.Scanner` |
| `strings` | 字符串操作 | `strings.Join`, `strings.Split` |

### 数据编码

| 包 | 用途 | 示例 |
|----|------|------|
| `encoding/json` | JSON 处理 | `json.Marshal`, `json.Unmarshal` |
| `encoding/xml` | XML 处理 | `xml.Marshal` |
| `encoding/base64` | Base64 编码 | `base64.StdEncoding` |

### 网络与 HTTP

| 包 | 用途 | 示例 |
|----|------|------|
| `net/http` | HTTP 客户端/服务器 | `http.Get`, `http.HandleFunc` |
| `net/url` | URL 解析 | `url.Parse` |

### 时间与日期

| 包 | 用途 | 示例 |
|----|------|------|
| `time` | 时间处理 | `time.Now()`, `time.Sleep` |
| `strconv` | 类型转换 | `strconv.Atoi`, `strconv.Itoa` |

## 🔤 fmt 格式化

```go
fmt.Printf("%v\n", 42)        // 42（默认格式）
fmt.Printf("%T\n", 42)        // int（类型）
fmt.Printf("%d\n", 42)        // 42（十进制）
fmt.Printf("%x\n", 42)        // 2a（十六进制）
fmt.Printf("%.2f\n", 3.14)    // 3.14
fmt.Printf("%s\n", "hello")   // hello
```

## 🌐 net/http 示例

```go
// HTTP 服务器
http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, %s!", r.URL.Path[1:])
})
http.ListenAndServe(":8080", nil)

// HTTP 客户端
resp, err := http.Get("https://api.example.com/data")
defer resp.Body.Close()
data, _ := io.ReadAll(resp.Body)
```
